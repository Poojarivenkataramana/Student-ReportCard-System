# 🎓 Student Report Card System (Python)

## 📌 Project Overview
This is a menu-driven Python application that helps manage and analyze student records.  
It allows users to add student details, view reports, and identify highest and lowest performers.

---

## 🚀 Features
- ➕ Add student name and marks  
- 📋 Display all student records  
- 🥇 Find highest marks student  
- 📉 Find lowest marks student  
- 🎯 Automatic grade calculation (A+, A, B+, B, C, D, F)  
- 🔄 Menu-driven interactive system  

---

## 🛠️ Technologies Used
- Python (Core concepts)
- Lists, Tuples
- Conditional Statements
- Loops

---

## 📂 How It Works
1. Run the program  
2. Choose options from menu:
   - Add Student  
   - Show Students  
   - Highest Marks  
   - Lowest Marks  
   - Exit  
3. System processes and displays results  

---

## 📊 Grading System
| Marks Range | Grade |
|------------|------|
| > 90       | A+   |
| 80 - 90    | A    |
| 71 - 79    | B+   |
| 66 - 70    | B    |
| 51 - 65    | C    |
| 35 - 50    | D    |
| < 35       | F    |
